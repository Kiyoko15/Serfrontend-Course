Html

HTML is the most basic building block of the web. It is responsible for defining the meaning and structure of web content. Furthermore, along with HTML, they are usually accompanied by other languages, for example CSS, which is used to describe the appearance/presentation of the website. In addition, there is also JavaScript which is responsible for the functionality/behavior of a web page.

Css

CSS is a style language (en-US) used to describe the presentation of a document written in HTML or XML (including several XML languages ​​such as SVG, MathML or XHTML). CSS describes how elements are displayed on screen, on paper, in speech, or in other media..

JavaScript

JavaScript® (sometimes abbreviated to JS) is a lightweight, interpreted, object-based language with first-class functions, best known as the scripting language for Web pages. It is also used in several other browserless environments, such as Node .js, Apache CouchDB and Adobe Acrobat. JavaScript is a prototype-based, multi-paradigm and dynamic language, supporting object-oriented, imperative and declarative styles (such as functional programming).
